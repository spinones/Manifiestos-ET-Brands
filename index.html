🧹 Empezar de cero

Para reiniciar todo: 1) borra primero las filas en tu Google Sheets (deja la fila de encabezados). 2) recién después usa este botón en cada equipo. Si lo haces al revés, los datos de prueba se vuelven a bajar de la planilla.

📥 Cargar histórico (N° de seguimiento)

Sube los números ya despachados antes. Quedan en la planilla como un manifiesto "Histórico" y sirven para que el sistema avise si alguno se vuelve a escanear. No necesitan tener chofer ni courier.

🗂 Cómo conectar Google Sheets (una sola vez)

1. Crea o abre tu planilla de Google Sheets.
2. Ve a Extensiones → Apps Script.
3. Borra el código existente y pega el código de abajo.
4. Guarda (Ctrl+S) → Implementar → Nueva implementación.
5. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier persona.
6. Copia la URL /exec y pégala en el campo de arriba.
